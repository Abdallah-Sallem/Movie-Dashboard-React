# Contributing

Thanks for considering contributing! A few guidelines to help your change
get merged quickly.

1. Fork the repository and create a feature branch from `main`:

```bash
git checkout -b feat/your-feature
```

2. Install dependencies and run the dev server locally:

```bash
cd React-Project
npm install
npm run dev
```

3. Keep commits focused and descriptive. Rebase or squash before opening a PR.

4. Run linting before pushing (optional but recommended):

```bash
npm run lint
```

5. Open a Pull Request describing the problem, your solution, and any testing
   steps. Link related issues if applicable.

6. Be responsive to review comments — maintainers may request changes.

Thanks — maintainers
